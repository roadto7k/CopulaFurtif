CopulaFurtif User Manual

These pages constitute the CopulaFurtif manual.

CopulaFurtif is a public tool for making copula analysis on trading data.

It can estimate copula’s best parameters for any kind of distribution, using scipy minimization problem. It can also be used to fine tuned copula’s estimation based on ML techniques.
